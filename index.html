Cómo incrustar archivos de Google Drive (ES / EN):
1) En Drive, abre el archivo → Compartir → “Cualquier persona con el enlace” (al menos Visor).
2) Copia el ID del archivo desde la URL: https://drive.google.com/file/d/FILE_ID/view.
3) Para video/PDF incrustado usa: https://drive.google.com/file/d/FILE_ID/preview en el iframe.
4) Para enlace de descarga/visualización directa, puedes usar la URL de “Compartir” normal.
